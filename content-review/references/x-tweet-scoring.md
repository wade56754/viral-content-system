> 本模块叠加在通用评分体系之上，替换/补充 X 平台专用维度

# X 推特专用评分模块

---

## 设计原则

**纯门禁，不当教练。**
- 只输出：分数 + 过/不过
- 不输出：怎么改、改哪里、具体建议
- 写手被打回后必须自己想办法，不给拐杖

---

## X 长推 / Article（100分制）

### 通用维度（55分）

| 维度 | 权重 | 核心判断 |
|------|------|---------|
| 🪝 钩子力 | 20分 | 前 130 字符（约65中文字）是否命中 4 种钩子之一：反直觉 > 数据 > 冲突 > 提问 |
| 📐 裂隙结构 | 10分 | 是否符合三幕结构（撕裂15-20% → 填充60-70% → 缝合15-20%）？ |
| 🫁 人味儿 | 10分 | 节奏像心跳？≥5处独占一行？零胶水句？有没有"我"的真实经历？ |
| 💥 争议/反常识 | 10分 | 有没有一句话让人想反驳？ |
| 🎯 行动引导 | 5分 | 结尾是否引导评论或书签（不是"关注我"）？ |

### X 专用维度（45分）

| 维度 | 权重 | 核心判断 | 数据来源 |
|------|------|---------|---------|
| 💬 互动设计 | 15分 | 有几处"评论触发器"（提问/争议/留白）？算法权重 Reply×13.5 | benchmarks.md |
| 📑 书签价值 | 10分 | 有几处可收藏素材（清单/公式/模板/数据）？Bookmark×10 | benchmarks.md |
| 🔗 外链合规 | 5分 | 正文零外链？链接是否放评论区？非Premium外链=-50%~-94% | benchmarks.md |
| 📏 字数区间 | 5分 | 是否在最佳传播区间（800-1500字符 / 中文400-750字）？ | 调研数据 |
| 🧲 自我反驳 | 10分 | 有没有"但你可能会说…"段？（裂隙理论核心，也是评论触发器） | x-longform-writer |

### 档位标准

| 档位 | 分数 | 判定 |
|------|------|------|
| 🔥 爆款潜力 | ≥ 85 | ✅ 直接发 |
| ✅ 合格 | 70-84 | ✅ 可以发 |
| ❌ 不合格 | < 70 | ❌ 打回重写，不给理由 |

### 维度最低线（任一不达标 → 打回，无论总分）

| 维度 | 最低线 | 理由 |
|------|--------|------|
| 🪝 钩子力 | < 12 | 前130字救不了 = 没人展开 |
| 💬 互动设计 | < 8 | 零互动 = 算法判死刑 |
| 🫁 人味儿 | < 5 | AI味太重 = 被取关 |
| 🔗 外链合规 | < 3 | 有外链 = 被降权 |

## 减分项清单（直接扣分，叠加在评分之后）

| 检测项 | 扣分 | 说明 |
|--------|------|------|
| 正文含外链 | -15 | 2025后外链=死刑 |
| 布道者姿态（"你应该…""你要…"） | -10 | 改用二传手："我花了X才想明白" |
| 纯焦虑结尾（不带赋能） | -8 | 焦虑+赋能才能互动2-3倍 |
| 超过 3 个 hashtag | -5 | 广告感太强 |
| 第一句 > 15字 | -3 | [作者]风格铁律：第一句≤15字 |
| "干货""建议收藏""建议转发" | -5 | 营销号信号词 |

## 审查输出格式（门禁模式）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚦 X 推文门禁 | {x-thread / x-short / x-article}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
字数：{N}字符（{中文N字}）

各维度：
{维度名}：{XX}/{权重}
{维度名}：{XX}/{权重}
...
减分项：{-N}（{原因}）

总分：{XX}/100
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
判定：✅ 合格，可以发 / ❌ 不合格，重写
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**⚠️ 不合格时：只输出"❌ 不合格，重写"。不说哪里不好，不给改法。**
**✅ 合格时：只输出"✅ 合格，可以发"。不加"但如果能改XXX就更好"。**
**🔥 爆款潜力时：输出"🔥 爆款潜力，直接发"。**
