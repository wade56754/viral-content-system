# 各平台审稿特化规则

## X Article / Thread

| 规则 | 说明 |
|------|------|
| 外链位置 | 必须放评论区（正文外链降 30-50% 曝光） |
| Hashtag | 0-1 个（≥4 负优化） |
| 即时可用性 | 权重最高（20分）：≥3 处可复制素材 |
| Thread 单条 | 200-280 字，每条独立有价值 |

## 小红书

| 规则 | 说明 |
|------|------|
| AI 辅助标注 | 必须标注（未标注 = KILL） |
| SEO 命中度 | 标题含核心词 + 前 100 字 + 标签覆盖 + 评论区长尾词 |
| 封面图指令 | 质量单独评分 |
| CTA 引评论 | 权重 ×4（CES 算法） |
| Emoji | 可多用（≤15 个） |
| 标签 | ≤10 个 |

## X 短推

| 规则 | 说明 |
|------|------|
| 评分制 | 60 分制换算百分制 |
| 最优长度 | 71-100 字符 |
| 核心权重 | 转推 ×20 权重 > 其他一切 |
| 独立性 | 必须独立成立（脱离上下文有意义） |

## 微信公众号

| 规则 | 说明 |
|------|------|
| 外链 | 无限制 |
| 长度 | 不限，但 3000-8000 字最优 |
| 标题 | 不超过 64 字符 |
| Hashtag | 无要求 |

## 通用规则

### 各平台最优字数区间

| 平台 | 最优字数 |
|------|---------|
| X 短推 | 71-100 字符 |
| X Article / Thread（单条） | 200-280 字 |
| 小红书 | 300-1000 字 |
| 微信公众号 | 3000-8000 字 |

### Emoji 上限

| 平台 | 上限 |
|------|------|
| X（短推 / Article） | ≤3 |
| 小红书 | ≤15 |
| 微信公众号 | ≤5 |

### 外链规则

| 平台 | 规则 |
|------|------|
| X | 正文外链降 30-50% 曝光，必须放评论区 |
| 小红书 | 正文不可放外链 |
| 微信公众号 | 无限制 |
