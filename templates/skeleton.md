## 基本信息

- **核心论点**：（一句话，≤20字）
- **裂隙类型**：🧠认知 / 🔧能力 / 💡信息 / ❤️情感
- **选题类型**：A经历驱动 / B观点驱动 / C工具驱动
- **展开模式**：模式A递进式否定 / 模式B并列打透
- **目标画像**：A AI焦虑者 / B 副业探索者 / C 创业老鸟 / D 内容人
- **收束风格**：行动指令 / 金句余韵 / 评论区引爆 / 个人迁移
- **核心比喻**：（什么比喻？引入/回扣/收束位置？）
- **自我反驳**：（放在哪里？最强反对意见是什么？）
- **二传手引用**：（引用谁的观点？放在哪里？）

## 论证结构

### 第一幕：撕裂
- **入场方式**：（个人故事/反常识/痛点具象化/WOW事实）
- **锚定**：（让读者点头的共识）
- **撕开**：（用什么打破锚定？）
- **承诺**：（"我花了X才想明白"）
- **节奏标签**：🪝钩子 → 📖故事/📊证据 → 💡观点

### 第二幕：填充

**论据1**：（一句话）
  - **先选故事**：（什么故事？走向本身能说明这个论据吗？）
  - **融合手法**：手法1故事即论证 / 手法2三明治 / 纯论证段
  - **读者误判**：（读者会在故事中途形成什么错误判断？）
  - **接住方式**：（金句？直接切下段？绝不用"所以这告诉我们"）
  - **无关细节**：（什么画面感细节？跟论点无关但让场景活了）
    → 检验：删掉这个细节，信息量有损失吗？没有 = 合格（说明它纯粹是画面感）
    → 检验：这个细节能调动哪个感官？（视觉/听觉/触觉/味觉/嗅觉）至少一个
    → 罗振宇级示例：宋徽宗过生日递酒被拒 / 75岁追寡妇 / 洗窗车
    → ❌ "他很紧张"（这是情绪描述不是感官细节）
    → ✅ "他手里的咖啡已经凉了"（触觉+视觉）
  - **打透方式**：（经历/数据/类比/反面？）
  - **弹药需求**：（需要什么支撑？来自哪个弹药文件？）
  - **🪝中段钩子**：（这段开头的 mini-hook 是什么？）
  - **🚪段尾 exit hook**：（这段结束时用什么把读者推向下一段？）
    → 检验标准：这句话让读者"想往下翻"而不是"点头认同"
    → ✅ 制造悬念："但这还不是最狠的。"
    → ✅ 制造不适："问题在于你已经出不去了。"
    → ✅ 抛出新矛盾："如果连能力都是庞氏，那什么不是？"
    → ❌ 总结句："所以XX很重要。"（让人点头=让人退出）
    → ❌ 不写（段落自然结束=读者自然退出）
  - **论证完整性检验**：
    → ① 读者能用自己的话跟朋友复述这个论据吗？不能 = 没论证完
    → ② 如果有故事，故事讲完整了吗？（有人物+有转折+有结局，不是提了一嘴就跑）
    → ③ 论据和核心论点的连接，读者自己能接上吗？不需要你说"所以这说明"
  - **节奏标签**：🪝→📖→📖→💡→🎯
  - **情绪温度**：🧊/🌡️/🔥

**校验**：
- [ ] 📖故事标签占比 ≥ 20%？
- [ ] 每10段至少1个🪝？
- [ ] 纯论证段（无故事）≤ 1个？
- [ ] 温度曲线不连续同温度？
- [ ] 每个论据段都有段尾 exit hook？（不是总结句）

- [ ] **前300字有具体场景？**（有人物+动作+感官细节）
