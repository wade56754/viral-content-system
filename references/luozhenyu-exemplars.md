## 案例1：《岳飞为什么必须死》（长视频版·40分钟级）

**类型：** A类认知颠覆（历史×权力结构）
**核心裂隙：** 读者以为岳飞死于奸臣/昏君 → 实际死于僵化的政治结构

### 原文结构拆解

**Hook（个人故事开头）：**
> "小时候听《岳飞传》觉得那个评书真的很迷人啊，我跟我老爹第一次发生冲突就在听这个评书上。"

👆 用自己小时候挨嘴巴的故事开头，不是上来就讲道理。

**第一层反常识（推翻表面答案）：**
- 表面：秦桧害死岳飞
- 进一步：高宗怕迎二圣还朝
- 再进一步：**高宗根本不怕二圣还朝**（用4个历史案例论证）

👆 关键技法：**连续翻转**。每次读者以为到底了，再翻一层。

**四个论证案例的编排（二传手姿态）：**
1. 朱元璋和小明王 → 没军队的皇帝毫无威胁
2. 宋徽宗回京后被软禁 → 亲爹都斗不过儿子
3. 宋徽宗过生日递酒被拒 → 一个细节让人心酸又信服
4. 岳飞自己改"迎二圣"为"迎天眷" → 岳飞也没那么傻

👆 关键技法：**不直接说"所以高宗不怕"，而是用4个故事让读者自己得出结论。**

**第二层反常识（真正的死因）：**
- 岳飞没有想象中那么能打（引朱熹、赵翼、钱大昕、胡适的判断）
- 猜忌才是根因：引《说郛》"猜忌天下之乱源也"
- 引《三体》黑暗森林法则作类比

👆 关键技法：**跨领域类比**（科幻小说→权力逻辑），**借他人之口**（朱熹说…赵翼说…）

**第三层（性格+结构的双重绞杀）：**
- 岳飞性格倔（三个具体事例）
- 但性格不是根因 → 宋代政治失去弹性才是
- 王安石变法后的党争 → 主战vs主和只能杀一方

**结尾升华：**
> "一千多年之后，我们再来看岳飞的这个悲剧，其实不是哪个人的悲剧……他是一个有才华的个人跟整个僵化的政治结构之间的悲剧。"

### 可学习的核心手法
1. **个人故事开头** → 小时候挨嘴巴（完全无关但建立情感连接）
2. **连续翻转** → 不止翻一层，翻三层，每层都有案例
3. **二传手姿态** → 朱熹说…赵翼说…《三体》里说…从不直接宣判
4. **无关细节注入** → 宋徽宗过生日那杯酒、"老拙"的署名（增加可信度）
5. **跨领域类比** → 三体黑暗森林→权力猜忌（降维打击读者认知）

## 案例2：《你因挣钱而伟大》（长视频版·40分钟级）

**类型：** A类认知颠覆（商业×宗教×人物传记）
**核心裂隙：** 读者以为伟人 = 道德完美 → 实际是"不伟大"才让普通人可模仿

### 原文结构拆解

**Hook（民间故事开头）：**
> "很多中国人都听说过这么一个民间传说，叫父子骑驴……这个故事的原创作者是一个两百年前的美国人。"

👆 用一个人人都知道的故事开头，翻转出处。

**悖论堆叠（制造好奇心）：**
连续7个"好奇怪"：
- 没当过总统，却印在面值最大的钞票上
- 不是革命元老，却是"建国之父"
- 不是大科学家，却发明了一堆鸡零狗碎的东西
- 不是大富翁，却被卡内基和梅隆奉为精神导师
- 不是文学家，却被马克·吐温奉为导师
- 私德有问题（私生子、勾搭朋友女友、75岁追寡妇）
- 杰斐逊说："我崇拜三个人，全是他"

👆 关键技法：**悖论堆叠**。不解释，只堆叠矛盾，让读者好奇到忍不住往下读。

**破题（一句话）：**
> "富兰克林他之所以拥有这样的地位，就因为他不伟大，所以他成为每一个普通美国人可以效仿的一个榜样。"

**论证路径（二传手+故事融合）：**
- 讲富兰克林家穷的细节（食不果腹但要假装田园诗般美好）
- 不信上帝的证据（"你干嘛老祈祷？向食物桶祈祷还省时间"）
- 《穷理查历书》的核心理念（又能挣又不花 = 美德）
- 马克斯·韦伯《新教伦理》的理论支撑
- 加尔文教派的逻辑缺环 → 富兰克林补上了

👆 关键技法：**思想史叙事**。不是说"挣钱是对的"，而是讲清楚历史上为什么人们觉得挣钱是错的，然后富兰克林如何翻转这个认知。

### 可学习的核心手法
1. **悖论堆叠** → 连续制造认知冲突，不急着解答
2. **人物缺陷不回避** → 私生子、勾搭朋友女友等全部摊开（增加可信度）
3. **思想史叙事** → 不是"我告诉你挣钱好"，而是"历史上人们为什么纠结"
4. **借权威之口** → 杰斐逊说…韦伯发现…马克·吐温说…

## 案例3：《曼哈顿生长的规则》（日播版·8分钟级）

**类型：** A类认知颠覆（城市规划→规则设计）
**核心裂隙：** 限制 ≠ 束缚 → 好的限制反而激发创造

### 原文结构拆解

**Hook（反常识现象）：**
> "曼哈顿地价那么贵，寸土寸金……但是在曼哈顿的街上常年能看到阳光。这个现象可不简单，是需要解释的。"

👆 关键技法：**把一个日常现象标记为"需要解释"**，瞬间制造好奇。

**时间线叙事（4个里程碑）：**
1. 1915年 恒生大楼 → "丑陋的地标"（反面教材）
2. 1916年 区划法案 → "越高越小"（简单规则）
3. 1952年 利华大厦 → 主动只占25%+花园（创造性解读规则）
4. 1961年 奖励法案 → 退让公共空间奖20%面积

👆 关键技法：**时间线叙事中每个节点都是一次"翻转"**。限制→反而更美→主动退让→奖励退让。

**无关细节注入（增加可信度）：**
- "利华公司的产品是清洁剂，他们就在大厦外部安了一辆洗窗车"
- "参与设计了这座大楼以后，利华公司的总裁还辞职了"

**结尾升华（跨领域验证）：**
- 戏剧的"三一律" → 限制产生了伟大作品
- 鸟群只需跟踪6-7只临近的鸟 → 简单规则产生复杂智能

**最终金句：**
> "好的规则有什么特点？两个特点：第一，极其简单，第二，边界清楚。"

### 可学习的核心手法
1. **日常现象→标记为需要解释** → 0成本制造好奇
2. **时间线叙事** → 每个节点一次翻转，读者跟着时间走
3. **无关细节** → 洗窗车、总裁辞职（不重要但让人记住）
4. **跨领域验证** → 建筑→戏剧→鸟群（同一原理三次验证=铁证）
5. **结尾极简** → 一句话总结，不啰嗦

## 三篇共性提炼（写作时必须对照）

| 维度 | 罗振宇的做法 | 反模式 |
|------|-------------|--------|
| **开头** | 个人故事/日常现象/民间传说 → 绝不从道理开始 | ❌ "今天我要谈谈关于XX的问题" |
| **观点表达** | 借故事浮现/借他人之口/用疑问引导 | ❌ "所以你应该…""道理很简单…" |
| **论证** | 多个独立案例指向同一结论，让读者自己推导 | ❌ 一个案例+大段解释 |
| **细节** | 插入无关但生动的细节（挨嘴巴/75岁追寡妇/洗窗车） | ❌ 每个细节都"有用"、都服务于论点 |
| **翻转** | 至少2-3层，每层都有新案例 | ❌ 只翻一层就开始总结 |
| **结尾** | 一句话金句/升华到更大框架 | ❌ 总结前文/重复论点 |
| **语气** | 口语化、聊天感、"你看""你想""对吧" | ❌ 书面语、论文感 |

## 使用方法

写作前（Phase 0-1）：
- 确认选题属于哪个案例类型（历史翻转/人物重塑/现象解谜）
- 选择对应案例作为结构参考

骨架设计（Phase 2）：
- 对照案例的翻转层数、故事编排、二传手姿态

写作中（Phase 4）：
- 每写完一个段落，对照"三篇共性表"自查

自检（Phase 6）：
- "如果罗振宇来写这个主题，他会怎么开头？" → 对比自己的开头
