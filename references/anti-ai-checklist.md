# 去AI化检测清单

> 按需加载：Pass 2 去AI化时读取
> 基于 VOICE_DNA v1.1 的24模式检测 + 呼吸感规则

---

## 24种 AI 味模式检测

每条逐一检查，命中 → 必须改。总命中数 ≤ 3 才合格。

### 结构类
1. **段落等长** — 连续3段以上长度相近 → 拆长补短
2. **匀速节奏** — 全文没有急缓变化 → 加一句话段/加留白
3. **对称结构** — 论据1-2-3像复制粘贴 → 每个论据换不同打透方式
4. **清单感** — 看起来像PPT大纲 → 加叙事过渡

### 用词类
5. **胶水句** — "接下来""值得注意的是""需要强调的是" → 全删
6. **官话腔** — "综上所述""不难看出" → 换成口语
7. **排比句** — 三段排比 → 保留最强的一句，其余删
8. **过度修饰** — "极其重要的""至关重要" → 砍掉形容词
9. **万能转折** — "然而""不过""但是"密度过高 → 替换为"但你有没有想过"
10. **概念堆砌** — 连续3个抽象概念无展开 → 中间插具体例子

### 开头类
11. **泛泛开场** — "在这个XX的时代" → 直接说事/亮数据
12. **自问自答** — "你有没有想过？答案是…" → 直接给答案
13. **假设问句** — "如果我告诉你…" → 直接说

### 逻辑类
14. **因果过度** — "因此""所以""由此可见"太多 → 减少因果连接词
15. **总结重复** — 结尾重复前文要点 → 找最近的出口走人
16. **预告内容** — "下面我们来看看" → 直接看

### 情感类
17. **鸡汤收尾** — "每个人都有无限可能" → 换成具体行动指令
18. **空洞鼓励** — "加油你可以的" → 删掉或给具体方法
19. **情绪词堆砌** — "震惊""绝了""太强了" → 靠内容说服

### 语气类
20. **布道者姿态** — "你应该""你需要" → 换成"我花了X才想明白"
21. **教科书口吻** — "本文将探讨" → 直接探讨
22. **全知视角** — "事实上""众所周知" → 删掉
23. **过度谦虚** — "个人浅见""我也不是专家" → 直接说观点
24. **万能呼吁** — "让我们一起…" → 具体到"你"

## 呼吸感规则

AI 写的文字 vs 人写的文字，最大区别 = 呼吸感。AI 匀速跑，人有心跳。

### 必须做到
- **短句断句**：一个逗号可以是一个段落
- **关键句独占一行**：前后空行，制造视觉停顿（全文≥5处）
- **禁止匀速节奏**：连续3段以上相近长度 = 机器味
- **逗号当句号**：制造"话没说完"的悬念感（全文≥3处）
- **砍掉过渡胶水句**：全删

### Wade 常用断句模式
- 「痛苦至极。」— 两字/四字独占一行
- 「后来我治好了，」— 逗号结尾，下一行才说怎么治的
- 「但是。」— 转折词独占一行

### 自检方法
朗读全文。换不了气的地方就要断。读起来像念稿 → AI 味重。读起来像聊天 → 对了。

|-----------|-----------|------|

## 量化标准

- AI味得分 = 24项中命中的数量
- **≤ 3 分**：✅ 合格，可发布
- **4-6 分**：⚠️ 需修改命中项
- **≥ 7 分**：❌ 重写 Pass 2
