## x-longform-writer 检索协议（v4）

### 何时调用
- **Phase 2（骨架搭建）** → 故事选择 + 论据溯源

### 两类素材的使用场景

| 需要 | 查哪里 |
|---|---|
| 完整故事（有人物/情节/转折） | 故事库：`*-details.md` + `*-by-scene-*.md` |
| 论据/数据支撑 | 弹药库：`*-weapons-论据.md` + `*-weapons-案例.md` |
| 开头金句/Hook | 弹药库：`*-weapons-金句.md` + `*-weapons-反常识.md` |
| 思维框架/模型引用 | 弹药库：`*-weapons-模型框架.md` |
| 论点参考/立论方向 | 弹药库：`*-weapons-论点.md` |
| 反直觉开头/制造冲突 | 弹药库：`*-weapons-反常识.md` |

### 检索流程

```
Step 1: 明确需求
  - 需要故事还是弹药？（还是两者都要？）
  - 场景/情绪/主题关键词？
  - 目标：论证支撑 / 开头引入 / 金句嵌入 / 框架引用？

Step 2: 选库（可多库混搭）
  - 商业/副业/变现 → 生财 scys-cases.md
  - 科学/心理/认知 → 万维钢
  - 历史/权力/人性/博弈 → 罗辑思维
  - 历史/哲学/文学/国学 → 熊逸
  - 经济/市场/制度/理性 → 薛兆丰
  - 金融/投资/财富/宏观 → 香帅
  - 思想/自由/正义/现代性 → 刘擎

Step 3: 读取对应文件
  【故事】→ {来源}-by-scene-{场景}.md → 找候选 ID → 读 {来源}-details.md
  【弹药】→ {来源}-weapons-{类型}.md → 直接按关键词搜索

Step 4: 选定 + 标记来源
  - 故事标注 ID（如 XY-0042）
  - 弹药标注 ID（如 XY-W0123）
  - 论据溯源：🟢 库内素材 = OK
```

### 匹配优先级

4. **无匹配**：用 writer 自身知识，标注"非库内素材"
